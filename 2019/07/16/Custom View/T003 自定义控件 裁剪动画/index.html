T003 自定义控件 裁剪动画

发表于 2019-07-16 | 分类于 Custom View |

参考知识点：01.5 精通自定义 View 之绘图基础——Canvas

一、原理

这个动画的原理很简单，就是每次将裁剪区域变大，在裁剪区域内的图像就会显示出来，而裁剪区域之外的图像不会显示。而关键问题在于如何计算裁剪区域。

从图示中可以看出，有两个裁剪区域。
